пока ты занят другим

Никаких ручных обновлений. Stellalink подключается к твоим аккаунтам и отражает реальное состояние прямо сейчас.

⚡
Живые данные
Steam, Faceit, GitHub, Last.fm, Spotify — данные тянутся через API и кэшируются в Redis. Профиль актуален всегда.
⠿
Drag-and-drop структура
Выбирай блоки и их порядок. Меняй расположение перетаскиванием — без кода и лишних настроек.
🎨
Кастомизация
Material 3, Apple-like, Fluent — пресеты через CSS-токены. Плюс свой CSS для полного контроля внешнего вида.
🔍
SEO / SSR
Nuxt SSR делает профиль индексируемым. Правильные мета-теги, OG-превью и быстрая загрузка из коробки.
📱
Мобильный первый
Профиль отлично выглядит на телефоне. Ссылку можно кинуть в Telegram — OG-превью сгенерируется автоматически.
🔒
Приватные блоки
Выбирай что публично, а что только для себя. Статистика просмотров и уникальных гостей в дашборде.
